FlexGrid
PHP Builder

Fluent, type-safe API for generating CSS Grid and Flexbox layouts — with presets, named areas, responsive breakpoints, and inline styles.

Installation

04 Gallery with Line-Based Spans
PHP
GridBuilder::make('.gallery')
  ->repeatColumns(4, GridValue::fr(1))
  ->gap('1rem')
  ->item(
    GridItem::select('.feature')
      ->area(GridArea::at(1, 1)
        ->spanColumns(2)
        ->spanRows(2))
  )
  ->build();
CSS output
.gallery {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 1rem;
}

.feature {
  grid-row: 1 / span 2;
  grid-column: 1 / span 2;
}

05 Responsive Breakpoints
PHP
GridBuilder::make('.layout')
  ->columns('1fr')
  ->gap('1rem')
  ->responsive(640, fn(GridBuilder $g) =>
    $g->columns(GridValue::repeat(2, GridValue::fr(1)))
  )
  ->responsive(1024, fn(GridBuilder $g) =>
    $g->columns(GridValue::repeat(3, GridValue::fr(1)))
      ->gap('1.5rem')
  )
  ->build();
CSS output
.layout {
  display: grid;
  grid-template-columns: 1fr;
  gap: 1rem;
}
@media (min-width: 640px) {
  .layout {
    grid-template-columns: repeat(2, 1fr);
  }
}
@media (min-width: 1024px) {
  .layout {
    grid-template-columns: repeat(3, 1fr);
    gap: 1.5rem;
  }
}

06 Inline Style & GridValue Helpers
PHP
// Inline style attribute
$style = GridBuilder::make()
  ->columns('1fr', '2fr')
  ->gap('1rem')
  ->autoRows(
    GridValue::minmax('100px', 'auto')
  )
  ->toInlineStyle();

// GridValue helpers
GridValue::fr(1.5);
GridValue::minmax('200px','1fr');
GridValue::fitContent('300px');
GridValue::Auto->value;
Output
// $style =
"display: grid;
 grid-template-columns: 1fr 2fr;
 gap: 1rem;
 grid-auto-rows: minmax(100px, auto)"

// Use in HTML:
<div style="<?= $style ?>">

05 Reverse, noWrap and Gap Override
PHP
FlexBuilder::make('.rail')
  ->direction(FlexDirection::ColumnReverse)
  ->noWrap()       // nowrap
  ->gap('0.5rem')
  ->gap('1rem')    // last call wins
  ->build();
CSS output
.rail {
  display: flex;
  flex-flow: column-reverse nowrap;
  gap: 1rem;
